# Dark_Forest_Exploration

WEBGL Javascript website simple game, made in Spring of 2019 for Computer Graphics.

# Languages and Tools
- Java
- HTML
- WebGl

# Controls

|Keys  | Direction |
| :---:   | :-: | 
|   W     | Moves forward |
|   S     | Moves backwards |
|   A     | Turns left |
|   D    | Turns right |
|   Z    | Moves down |
|   X    | Moves Up |
